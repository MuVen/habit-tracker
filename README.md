# 🎯 Habit Tracker

A beautiful, production-grade habit tracker dashboard with smart scheduling, visual analytics, and Google Sign-In.

**[Live Demo →](https://vemunuku.github.io/habit-tracker/)**

## Features

- 🔥 **Streak Tracking** — Current & best streak at a glance
- 📅 **Smart Scheduling** — Daily, alternate days, or custom weekdays
- 📊 **Visual Analytics** — 30-day trend line chart, per-habit bar chart, contribution heatmap
- 🎨 **Customizable** — 10 colors, 20 icons, dark mode
- ✏️ **Full CRUD** — Add, edit, delete habits with time/duration
- 🔐 **Google Sign-In** — OAuth 2.0 authentication
- 🏠 **Stunning Landing Page** — Animated hero, features, CTA
- 💾 **Privacy First** — All data in localStorage, no server
- 📱 **Responsive** — Desktop, tablet, mobile

## Tech Stack

- HTML + Tailwind CSS (CDN)
- Vanilla JavaScript
- Chart.js for analytics
- Lucide Icons (SVG)
- Google Identity Services
- GitHub Pages hosting

## Getting Started

1. Clone the repo
2. Open `index.html` in a browser
3. Click "Try Demo" or sign in with Google

### Google Sign-In Setup (optional)

1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Create OAuth 2.0 Client ID (Web application)
3. Add your GitHub Pages URL as authorized origin
4. Replace `YOUR_CLIENT_ID` in `auth.js` and `index.html`
